# Accounts Receivable Dashboard (Power BI)

## Project Overview
This project presents an interactive Accounts Receivable dashboard developed in Power BI to help finance teams monitor outstanding invoices, analyze aging balances, and prioritize collection efforts.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key KPIs
- Total Accounts Receivable
- Current AR
- 31–60 Days
- 61–90 Days
- 91–120 Days
- Over 120 Days
- Percentage of AR >60 Days

## Dashboard Features
- Customer Analysis
- Project Manager Analysis
- Department Analysis
- Aging Summary
- Interactive Filters

## Business Value
The dashboard provides clear visibility into overdue receivables, helping identify high-risk customers and departments while supporting faster, data-driven collection decisions.
