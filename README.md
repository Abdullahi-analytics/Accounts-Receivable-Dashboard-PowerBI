# 📊 Accounts Receivable Dashboard
![Accounts Receivable Dashboard](accounts-receivable-banner.png)

> An interactive Power BI dashboard that provides comprehensive Accounts Receivable (AR) insights, enabling finance teams to monitor outstanding balances, analyze aging trends, identify high-risk accounts, and improve cash flow management.

## 📖 Project Overview

The Accounts Receivable Dashboard was developed to transform raw financial data into actionable insights. It provides a centralized view of receivables by customer, project manager, department, and aging category, allowing stakeholders to monitor payment performance and prioritize collection efforts effectively.

## 🎯 Business Objectives

- Monitor total Accounts Receivable balance.
- Analyze receivable aging across different time buckets.
- Identify customers with high overdue balances.
- Evaluate Project Managers based on outstanding receivables.
- Monitor departmental receivable performance.
- Improve collection efficiency and cash flow visibility.

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

## 📈 Dashboard Features

### KPI Cards

- % of Total Accounts Receivable (Customer)
- % of Over 60 Days AR (Customer)
- % of Total Accounts Receivable (Project Manager)
- % of Over 60 Days AR (Project Manager)
- % of Total Accounts Receivable (Department)

### Financial Summary

- Total Accounts Receivable
- Current Balance
- 31–60 Days
- 61–90 Days
- 91–120 Days
- Over 120 Days

### Customer Analysis

- Top 10 Customers with Over 60-Day Outstanding Balances
- Customer Treemap for Outstanding Receivables

### Project Manager Analysis

- Top 10 Project Managers with Over 60-Day Receivables

### Department Analysis

- Department-wise Outstanding Receivables
- Department Performance Table

### Interactive Features

- Dynamic Filters (Slicers)
- Interactive Cross-filtering
- Responsive Visualizations

## 💡 Key Insights

- Total Accounts Receivable exceeded **$56.2M**.
- Nearly **48.6%** of receivables were in the Current category.
- Approximately **34.5%** of balances fell within the 31–60 day aging period.
- Overdue receivables (more than 60 days) highlighted customers and departments requiring immediate attention.
- Customer-level analysis revealed the highest outstanding balances among key accounts, helping prioritize collection efforts.

## 📊 Business Impact

This dashboard enables organizations to:

- Monitor receivable performance in real time.
- Reduce overdue invoices through proactive collections.
- Identify high-risk customers and departments.
- Improve financial reporting and transparency.
- Support better cash flow forecasting.
- Enable data-driven decision-making.
